DIY RC Airplane from Aliexpress.

This project is about building a RC airplane from separate parts instead of buying a ready-to-fly model.

I don't have a 3D printer, so I will use a lightweight throwable foam plane.

I want to learn how RC airplanes work, how to install electronics, solder wires and set up the radio system.

The airplane will use a brushless motor, servos, radio receiver and transmitter and a LiPo battery.

I will assemble the airplane, install all electronics, configure the radio system, test everything, make the first flight and improve the setup if needed.

I will upload photos, videos and progress updates during the project
