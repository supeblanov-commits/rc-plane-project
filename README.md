Budget complete DIY FPV RC Airplane from Aliexpress.

This project is about building a FPV RC airplane from separate parts instead of buying a ready-to-fly model.

I don't have a 3D printer, so I will use a lightweight throwable foam plane.

I want to learn how RC airplanes work, how to install electronics, solder wires, set up the radio system, install an FPV camera and fly the airplane using FPV.

The airplane will use a brushless motor, ESC, servos, radio receiver and transmitter, FPV camera, video transmitter, FPV goggles and a LiPo battery.

I will assemble the airplane, install all electronics, configure the radio system, test everything, make the first flight and improve the setup if needed.

I will upload photos, videos and progress updates during the project so other people can see how it is being built and what I learn while working on it.
